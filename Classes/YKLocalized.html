Overview

Wrapper for localization calls, that caches localized key/value pairs.

    Current locale identifier with correct language. Normally, [[NSLocale currentLocale] localeIdentifier] only returns the locale identifier relevant to the region format. That is, for language = Español and region format = US, [[NSLocale currentLocale] localeIdentifier] == @“en_US”.
